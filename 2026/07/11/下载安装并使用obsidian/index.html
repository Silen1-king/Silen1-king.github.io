    下载安装并使用obsidian

    Posted on 2026-07-11 Edited on 2026-07-13 In 学习笔记

    obsidian是一款无比前卫的电子笔记软件，从零开始完成下载安装与基本使用。

    一、什么是 Obsidian

    Obsidian 是一款本地优先的知识管理工具，它的核心特点包括：

    • Markdown 编辑：所有笔记以纯文本 .md 格式存储，你可以用任何编辑器打开
    • 双向链接：通过 [[笔记名]] 语法轻松关联笔记，构建知识网络
    • 图谱视图：可视化展示笔记之间的关联关系
    • 丰富的插件生态：社区提供了海量插件来扩展功能
    • 完全离线：数据存储在本地，不上传云端，保护隐私
    • 免费使用：个人使用完全免费

    二、下载 Obsidian

    官网下载

    打开 Obsidian 官网：https://obsidian.md

    三、安装 Obsidian

    Windows 安装

    1. 双击下载好的 Obsidian-x.x.x.exe 安装包
    2. 选择安装路径（默认路径即可）
    3. 点击 “Install” 开始安装
    4. 安装完成后启动 Obsidian

    macOS 安装

    1. 打开下载的 .dmg 文件
    2. 将 Obsidian 图标拖入 Applications 文件夹
    3. 首次打开如提示安全警告，进入系统偏好设置 → 安全性与隐私允许运行

    四、创建笔记库（Vault）

    首次启动后，创建一个 Vault：

    1. 点击 “Create new vault”
    2. 输入库的名称，例如 “我的笔记”
    3. 选择存储位置（建议放在一个容易找到的文件夹）
    4. 点击 “Create”

    一个 Vault 本质上就是一个文件夹，所有 .md 文件就是你的笔记，可以随时在资源管理器中查看。

    五、界面介绍

    区域 功能
    左侧边栏 文件列表、搜索、书签
    中间编辑区 笔记的编辑和预览
    右侧边栏 大纲、反向链接、标签面板
    顶部工具栏 快速操作按钮

    按 Ctrl+E 切换编辑/阅读模式，按 Ctrl+P 打开命令面板。

    输入 [[ 时自动补全笔记名。按住 Ctrl 悬停链接可预览内容。

    代码块与图片

    七、图谱视图

    点击左侧边栏图谱图标（或 Ctrl+G）：

    • 每个圆点代表一篇笔记
    • 连线表示笔记之间的链接
    • 节点越大，被引用越多

    八、插件系统

    进入设置 → Community plugins开启社区插件。

    推荐必装插件：

    插件名 功能
    Calendar 日历视图，方便浏览日记
    Dataview 将笔记当作数据库查询
    Excalidraw 手绘风格图表绘制
    Templater 高级模板功能

    九、同步与备份

    • Obsidian Sync：官方付费同步，端到端加密
    • 第三方云盘：将 Vault 放在 iCloud、OneDrive、坚果云等文件夹中
    • Git 备份：使用 Git 搭配 GitHub 私有仓库

    总结

    Obsidian 的哲学：先记下来，再连接起来。

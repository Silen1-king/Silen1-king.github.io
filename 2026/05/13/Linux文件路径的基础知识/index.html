Linux文件路径的基础知识

Posted on 2026-05-13 Edited on 2026-05-21

EXAMPLES

/ 文件系统的“锚点”，读作“root（根）”。
/usr Unix 系统资源目录，包含所有系统文件。
/usr/bin 计算机上已安装程序的可执行文件目录。
/usr/lib 供计算机上的程序使用的共享库目录。
/usr/share 程序资源目录（图标、美术资源等）。
/etc 系统配置文件目录。
/var 日志、缓存等数据目录。
/home 用户所有数据的目录。
/home/hacker 在 pwn.college 环境中，你个人的数据目录。
/proc 运行时进程数据目录。
/tmp 临时数据存储目录。

三种命令

pwd：显示当前终端会话所处目录的绝对路径

cd：改变当前终端的目录，在文件系统中移动

  1. 切换到指定路径（绝对路径）cd /home/hacker
  2. 切换到相对路径（基于当前目录）cd Documents
  3. 回到上一级目录 cd ..
  4. 直接回到用户主目录（和 cd ~ 效果一样）cd
  5. 回到上一次所在的目录（来回切换超方便）cd -

ls：列出指定目录下的文件和子目录

  1. 列出当前目录的文件/目录（默认模式）ls
  2. 列出指定目录的内容 ls /home
  3. 显示详细信息（权限、大小、修改时间等）ls -l
  4. 显示所有文件，包括隐藏文件（以 . 开头的文件 ls -a
  5. 人类可读格式显示文件大小（KB/MB/GB ls -lh

路径的核心组成

符号/元素 含义 示例
/（开头） 表示这是绝对路径，从根目录开始定位 /home/yans/flags
/（中间） 路径分隔符，用于分隔目录层级 flags/TOPSECRET 中的 /
. 代表当前目录，写不写都可以，比如 ./TOPSECRET 和 TOPSECRET 效果相同 ./lib
.. 代表上一级目录，每写一个 .. 就向上退一层 ../../ 表示向上退两级目录
文件名 路径的终点，指向具体的文件 TOPSECRET

~

单独的~代表当前用户的家目录
~用户名代表指定用户的家目录

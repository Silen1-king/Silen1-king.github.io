# 归档、统计与全局搜索

## 归档

`/archives/` 使用原生 `details/summary` 按年份、月份折叠。归档分页已关闭，
因此主归档页始终能够展示全部历史文章。

## 博客统计

侧栏统计由 `scripts/archive-stats.js` 根据当前 Hexo 文章数据实时计算：

- 中文字符按一个字计算，英文和数字连续串按一个词计算；
- 技术栈比例按“带有该标签的文章数 / 总文章数”计算；
- 最多展示出现频率最高的 8 个标签；
- 领域概述使用真实的前 4 个标签，不使用预设分类。

## 全局搜索

搜索数据来自 `hexo-generator-searchdb` 生成的 `/search.json`，索引标题、标签、
分类与正文。点击页头搜索按钮，或按 `Ctrl+K` / `Cmd+K` 打开；按 `Esc` 关闭。

结果支持多关键词、相关度排序、匹配高亮，以及方向键选择和 Enter 打开。
